## Function

This weather app will first ask for permission to get your current location at the home page to get your current location and show you the weather forecast for that place.

You can also use the search bar to look for cities and get the weather report for all of the locations with those name, to differentiate it there is the location of the country along with the country flags.

as it only shows 3 country per page, there is pagination to show the next result of the search.

If u click on the down arrow of a searched country, it will show a more detailed overview of the weather report.
